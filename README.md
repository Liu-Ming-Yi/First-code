# 🤖 利用 AI 學習 HTML (Learning HTML with AI)

這個專案記錄了我如何透過 AI 模型（例如 Gemini, ChatGPT 等）作為輔助的工具，來學習、理解和實踐網頁開發中最基本的標記語言 HTML。

目標是探索 AI 在以下方面的潛力：
1.  **程式碼生成與解釋**：讓 AI 撰寫 HTML 並且達到我想要的效果。
2.  **主要目的**：利用 AI 協助完成小型 HTML 網頁的結構。
