# 🤖 利用 AI 學習 HTML (Learning HTML with AI)

> 專注於利用現代 AI 工具（如大型語言模型）來加速和優化 HTML 基礎知識的學習與實踐過程。

## 📖 專案簡介 (Introduction)

這個專案記錄了我如何透過 AI 模型（例如 Gemini, ChatGPT, Claude 等）作為輔助的工具，來學習、理解和實踐網頁開發中最基本的標記語言——HTML。

目標是探索 AI 在以下方面的潛力：
1.  **程式碼生成與解釋**：讓 AI 撰寫 HTML 並解釋每個標籤的作用。
2.  **ˋ主要目的**：利用 AI 協助完成小型 HTML 網頁的結構。
