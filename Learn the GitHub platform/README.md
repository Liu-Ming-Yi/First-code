# 🤖 利用 AI 學習 HTML (Learning HTML with AI)

> 專注於利用現代 AI 工具（如大型語言模型）來加速和優化 HTML 基礎知識的學習與實踐過程。

## 📖 專案簡介 (Introduction)

這個專案記錄了我/我們如何透過 AI 模型（例如 Gemini, ChatGPT, Claude 等）作為編程導師和助手，來學習、理解和實踐網頁開發中最基本的語言——HTML。

目標是探索 AI 在以下方面的潛力：
1.  **程式碼生成與解釋**：讓 AI 撰寫 HTML 範本並解釋每個標籤的作用。
2.  **錯誤排查**：請 AI 檢查和修正我/我們手動撰寫的 HTML 錯誤。
3.  **概念釐清**：針對 HTML 標籤、屬性、語義化等概念進行提問並獲取清晰的解答。
4.  **專案實踐**：利用 AI 協助完成小型 HTML 網頁的結構搭建。

## 💡 AI 互動技巧 (AI Prompting Tips)

在與 AI 學習 HTML 過程中，建議使用以下提示詞策略以獲得最佳效果：

* **請求解釋**：`「請詳細解釋 HTML 中的 <meta> 標籤及其常見屬性的用途。」`
* **請求範例**：`「請給我一個包含一個 `<audio>` 標籤和一個 fallback 內容的 HTML 範例。」`
* **請求除錯**：`「這段 HTML 程式碼無法顯示圖片，請幫我檢查並修正錯誤：[貼上程式碼]。」`
* **角色扮演**：`「請扮演一位嚴格的 Web 開發導師，批改我這段 HTML 結構是否符合語義化標準：[貼上程式碼]。」`
